EN:
Copy the files to a USB and run the InfoPC.exe program (It is optional to copy the program code)

This program includes cmd commands such as:
* systeminfo (Saves system information)
* whoami (pc name)
* driverquery (Driver information)
* ipconfig / all (Network and DNS information)
* net view (Information about the workgroup)
* sc query (Report the status of the secure channel the last time you used it)
* tasklist (Processes in progress at the time of program execution)
* ver (Windows version)
* netstat -ao (Returns basic statistics on all network activities)
* ipconfig / displaydns (Show dns)
* getmac (Get mac from device)
